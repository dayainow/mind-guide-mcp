<h1 align="center">마음길잡이 MCP</h1>

<p align="center">
  카카오 PlayMCP용 정신건강·복지 안내 MCP 서버<br/>
  "요즘 힘든데 근처 상담할 곳 있을까?" 한 마디로 상담기관과 복지제도를 연결합니다.
</p>

---

## 무엇을 하는 서버인가

마음길잡이는 **상담하는 봇이 아닌, 연결해주는 봇**입니다.

사용자가 카카오톡에서 지역과 상황을 말로 전달하면:
- 전국 정신건강복지센터 연락처·운영시간 안내 (공공데이터 기반, 163개 기관)
- 청년 바우처·산후 정서지원 등 맞춤 복지제도 12종 추천
- 위기 신호 감지 시 자살예방상담전화(109) 등 즉시 연락처 우선 안내

대화 내역을 읽거나 개인정보를 수집하지 않습니다. 사용자가 직접 입력한 내용만으로 동작합니다.

---

## 도구 (MCP Tools)

| 도구 | 설명 |
|------|------|
| `crisis_resources` | 위기 시 즉시 연락처 안내 (109, 1577-0199, 129, 1388, 119) |
| `find_counseling_centers` | 지역·상황별 정신건강복지센터 추천 |
| `recommend_welfare` | 상황·나이·가구형태별 복지제도 추천 |

> 모든 추천 도구는 위기 신호를 먼저 감지해, 신호가 있으면 추천보다 위기 연락처를 앞에 안내합니다.

---

## 대화 설계

단순한 정보 안내 봇이 아닌, 실제로 힘든 사람이 쓴다는 전제로 대화 흐름을 설계했습니다.

```
1단계 — 공감      힘들다는 말에 바로 도구를 쓰지 않고, 먼저 마음을 받아줍니다.
2단계 — 확인      "도움받을 수 있는 곳 찾아드릴까요?" 의사를 먼저 묻습니다.
3단계 — 안내      동의하면 지역·상황을 물어 상담기관 또는 복지제도를 안내합니다.
4단계 — 신청 안내 추천 후 "처음 연락할 때 이렇게 말해보세요" 스크립트를 함께 줍니다.
5단계 — 체크인    "다음에 또 얘기해요" 등으로 대화를 자연스럽게 마무리합니다.
```

**낙인 제거 메시지** — 추천할 때 아래 메시지를 자연스럽게 녹여 부담을 낮춥니다.

> "마음이 힘든 건 감기 걸린 것과 같아요. 치료받는 게 당연한 거예요."  
> "상담 받는다고 큰 문제가 있는 사람인 게 아니에요. 오히려 용기 있는 거예요."  
> "지금 연락하는 게 절대 늦지 않았어요."

---

## 설계 원칙 (안전)

1. **연결하는 봇** — 감정 진단·치료 조언 금지. 실제 기관으로 연결하는 역할만 합니다.
2. **위기 우선** — 위기 신호 감지 시 추천보다 즉시 연락처를 먼저 안내합니다.
3. **위기 수단·방법은 절대 다루지 않습니다.**
4. **AI가 사람 상담을 대체하지 않습니다.** 항상 실제 기관 연결을 권합니다.

## 로컬 실행

```bash
pip install -r requirements.txt
cp .env.example .env   # API 키 입력 (없어도 더미 데이터로 동작)

# → http://0.0.0.0:8000/mcp (streamable-http)
```
