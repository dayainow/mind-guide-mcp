# 마음길잡이 MCP — 프로젝트 컨텍스트 (Claude Code 인수인계용)

이 파일은 Claude Code가 이 프로젝트를 이어받기 위한 맥락 문서입니다.
새 세션을 시작하면 이 파일부터 읽고 작업하세요.

## 무엇을 만드는가
카카오 PlayMCP에 등록할 **정신건강·복지 안내 MCP 서버**.
"요즘 힘든데 근처 상담할 곳 있을까?" 같은 카카오톡 한 줄 질문에,
위치·상황을 말로 받아 상담기관과 복지제도를 안내한다.

## 어쩌다 이 아이템이 됐나 (중요한 제약)
원래는 "카카오톡 채팅방 대화로 소설/관계도 만들기"를 하려 했으나,
**카카오는 채팅방 대화를 읽어오는 API를 제공하지 않는다**(나와의 채팅방
'보내기'만 가능). 그래서 "대화를 읽지 않고, 사용자가 말로 주는 정보만으로
동작하는" 아이템으로 방향을 틀었다. 이 서버의 모든 입력은 사용자가 채팅에
직접 적는 값(지역, 상황 등)이며, 어떤 개인 데이터도 읽지 않는다.

## 절대 지켜야 할 설계 원칙 (안전)
이 분야는 실제로 힘든 사람이 쓸 수 있어 일반 도구보다 엄격하게 다룬다.
1. **상담하는 봇이 아니라 연결해주는 봇.** 감정을 진단("우울증 같아요")하거나
   치료 조언을 하지 않는다. 역할은 ① 짧게 마음을 받아주고 ② 실제 기관·제도로
   빠르게 연결하는 것뿐.
2. **위기 우선.** 추천 도구는 입력에서 위기 신호(죽고 싶다 등)를 먼저 감지하면
   추천보다 위기 연락처를 앞에 안내한다. (`_detect_crisis` → `_crisis_block`)
3. **위기 수단·방법은 절대 다루지 않는다.** 신호어 목록에도, 응답에도 포함 금지.
4. **AI가 사람 상담을 대체하는 척하지 않는다.** 항상 실제 기관 연결을 권한다.

## 현재 상태
- `mind_guide_server.py` — FastMCP 서버. 도구 3개 등록·동작 확인 완료.
  - `crisis_resources(region="")` — 위기 시 즉시 연락처 안내
  - `find_counseling_centers(region, situation="", topn=3)` — 상담기관 추천
  - `recommend_welfare(situation, age=None, household="", region="")` — 복지제도 추천
- 데이터는 전부 **더미**(`_CENTERS`, `_WELFARE`). 매칭은 키워드 기반.
- 전송 방식: `streamable-http`, 기본 `http://127.0.0.1:8000/mcp`. 로컬 기동 확인됨.

## 다음 할 일 (우선순위)
1. **위기 연락처 검증.** `CRISIS_CONTACTS`의 번호(자살예방 109, 정신건강
   1577-0199, 보건복지 129, 청소년 1388)가 현재도 유효한지 공식 출처로 재확인.
   사람 안전과 직결 — 틀리면 안 됨.
2. **실제 데이터 연동.** 보건복지부 공개 API(① 정신건강·심리상담 센터 정보,
   ② 맞춤형 복지서비스 정보) 신청·키 발급 후, `_CENTERS`/`_WELFARE`를 API
   호출로 교체. 응답 스키마에 맞춰 파싱 함수 작성. 키는 환경변수로 관리(.env),
   코드/깃에 하드코딩 금지.
3. **배포.** 공개 URL로 서버를 띄운다(예: Railway/Render/Fly, 또는 터널링).
   HTTPS 필수. `host=0.0.0.0`, 포트는 환경변수.
4. **PlayMCP 등록.** playmcp.kakao.com에서 카카오 로그인 → 원격 MCP 서버 URL로
   배포 주소(/mcp) 등록 → AI 채팅에서 도구 4개 동작 테스트.
5. (선택) 도구 추가: 지역 자동 추정, 카카오맵 MCP와 조합한 길찾기 등.

## 코딩 메모
- `pip install mcp` (이 환경에서 시스템 PyJWT 충돌 시 `--ignore-installed PyJWT`)
- 더미↔실데이터 교체가 쉽도록 데이터 소스 함수를 분리해두면 좋다.
- 응답 문구 톤: 짧고 담백하게, 단정·진단 금지, 실제 기관 연결로 마무리.
